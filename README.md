How to run:

1. Create and activate a virtual environment (Python 3.11):
   python -m venv venv
   On Windows:
     .\venv\Scripts\activate
   On Linux/macOS:
     source venv/bin/activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python run.py

4. Open in your browser:
   http://127.0.0.1:5000
